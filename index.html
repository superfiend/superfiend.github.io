将文章置顶的方法

将文章置顶需要执行以下两个命令:

  1. $ npm uninstall hexo-generator-index –save
  2. $ npm install hexo-generator-index-pin-top –save

随后在需要置顶的文章的头部信息中添加: top: true 就ok了。
