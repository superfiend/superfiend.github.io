http请求状态码

这篇文章主要记录http网络请求的一些状态码
  1. 200 -> 请求成功
  2. 403 -> 服务器禁止请求
  3. 404 -> 访问网页不存在
  4. 500 -> 服务内部不可用
  5. 503 -> 访问服务不可用
