hexo安装rss插件

本篇文章关于如何安装rss插件

  1. 进入blog目录, 执行npm install hexo-generator-feed
  2. 根目录下的_config.yml文件中，添加以下配置
    #RSS订阅
    plugin:
    - hexo-generator-feed
    feed:
    type: atom
    path: atom.xml
    limit: 20
  3. 在主题目录下的_config.yml目录下，添加如下配置
    rss: /atom.xml
