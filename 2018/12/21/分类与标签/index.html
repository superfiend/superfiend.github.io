分类与标签(landscape)

新建分类

  1. 新建分类页面
    hexo new page categories
  2. 给分类页面添加类型
    在source文件夹中的categories文件夹下找到index.md文件，并在它的头部加上type: “categories”。
  3. 给模板添加分类属性
    打开scarffolds文件夹里的post.md文件，给它的头部加上categories:。
  4. 给文章添加分类

新建标签

  1. 新建标签页面
    hexo new page categories
  2. 给标签页面添加类型
    在source文件夹中的tags文件夹下找到index.md文件，并在它的头部加上type: “tags”。
  3. 给文章添加标签
  4. 给文章添加多个标签的方法
    a. 直接采用[], 如[tag1, tag2]
    b. 前面加_, 如_tag1, _tag2

ps: 新建分类与新建标签是类似的, 不过post.md文件自带了tag属性, 若没有, 需要自行添加。
