今天学习了tkinter, 本来不想要记录下来的, 但是, 想到时间永远是记忆最大的敌人, 还是记录下来好了。

  1. 窗口对象 Tk -> tk = Tk()

  2. 窗口开始 tk.mainloop()

    a. Button -> btn = Button(text=’button’) # 按钮

    b. Label -> lbl = Label(text=’label’) # 标签

    c. Entry -> ent = Entry() # 文本输入框

    d. more…

  4. 窗口退出 tk.quit()

    a. pack()

    b. grid(row=?, column=?)
